
# Monthly Expenses Tracker PWA

## Features
- Firebase Authentication
- Firestore realtime sync
- Mobile responsive PWA
- Dark mode
- Export CSV
- Auto logout timer
- Installable app

## Setup

1. Create Firebase project
2. Enable:
   - Authentication (Email/Password)
   - Firestore Database
3. Replace firebaseConfig in app.js
4. Upload files to Firebase Hosting or Netlify
